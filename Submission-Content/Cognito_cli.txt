The following cli commands were used to create the cognito user pool and client used in the project

To create user pool:
aws cognito-idp create-user-pool --pool-name Group9UserPool --username-attributes email --auto-verified-attributes '[]'

To retrieve user pool id:
aws cognito-idp list-user-pools --max-results 50

To create client:
aws cognito-idp create-user-pool-client --user-pool-id us-east-1_222mvHFqQ --client-name messagingService --no-generate-secret

To retrieve client id of client:
aws cognito-idp list-user-pool-clients --user-pool-id us-east-1_222mvHFqQ
